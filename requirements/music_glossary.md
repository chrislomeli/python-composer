## üéº Composition Terms

| Term | Layman‚Äôs Explanation | How It Relates to MIDI |
|------|----------------------|-------------------------|
| **Movement** | Large, self-contained section (like a chapter). | MIDI does not encode movements; this is handled by DAW arrangement. |
| **Motif** | Short recurring musical idea. | A repeating MIDI pattern or clip. |
| **Phrase** | A musical ‚Äúsentence.‚Äù | Represented as grouped MIDI notes with natural boundaries. |
| **Theme** | A longer, central musical idea. | The main MIDI melody or chord progression. |
| **Harmony** | Notes played together forming chords. | Directly represented as simultaneous MIDI note-on events. |
| **Melody** | The main tune. | Implemented as sequential MIDI notes. |

| **Ostinato** | A repeating pattern. | A looping MIDI clip or repeated rhythmic figure. |

| **Texture** | Thickness or thinness of musical layers. | Achieved via track density and instrumentation choices. |

## üéß MIDI-Specific Sound Design Concepts

| Term | Layman‚Äôs Explanation | How It Relates to MIDI |
|------|----------------------|-------------------------|
| **Velocity** | How hard a note is played. | Directly encoded in MIDI; affects loudness/brightness. |
| **Aftertouch** | Pressure after a key is pressed. | Sends continuous modulation for vibrato, brightness, etc. |
| **CC (Control Change)** | Adjustable parameters in MIDI. | Controls volume, expression, modulation, filters, etc. |
| **Modulation (CC1)** | Broad expressive control. | Usually mapped to vibrato or brightness. |
| **Expression (CC11)** | Fine volume shaping. | Secondary volume control that doesn‚Äôt override the main volume (CC7). |
| **Articulation** | Playing style (staccato/legato/etc.). | Triggered via keyswitch notes, CCs, or program changes. |
| **Envelope / ADSR** | Attack, decay, sustain, release shape. | MIDI cannot define ADSR; it only triggers envelopes (note-on) or adjusts them if synth exposes CC/NRPN/SysEx. |
| **LFO** | Slow modulation source for vibrato, tremolo, etc. | MIDI cannot define LFO shapes/rates; it can only change their depth or speed through CC. |
| **Pitch Bend** | Smooth pitch adjustments. | Native MIDI message for continuous pitch shifting. |
